# Markdown笔记

## 基础知识

**字体加粗、斜体**

在加粗文字前后用 ****** 标记，在斜体文字前后用 ***** 标记,在加粗、斜体文字前后用 *********** 标记

**下划线、删除线**

将文字填入 **<u> </u>** 中即可加下划线

在添加删除线文字前后用 **~~** 标记

**引用**

在文字前面加上 **>**

**列表**

无序列表：在内容前面加上 **-  *  +** 

有序列表：在内容前面加上 **数字** + **.** + **空格** 的方式

**公式**

行内公式：在公式前后加上 **$**

独立公式：在公式前后加上 **$$**
